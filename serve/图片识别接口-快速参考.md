# 图片识别分类接口 - 快速参考

## 接口信息

- **地址：** `POST http://localhost:3000/recognize`
- **格式：** `multipart/form-data`
- **字段名：** `image`（文件字段）

## 成功响应

```json
{
  "category_id": 1,
  "category_name": "美食生鲜",
  "confidence": 0.95
}
```

## 错误响应

```json
{
  "error": "图片识别失败",
  "message": "无法识别图片内容，请手动选择分类"
}
```

## 微信小程序调用

```javascript
wx.uploadFile({
  url: 'http://localhost:3000/recognize',
  filePath: tempFilePath,
  name: 'image',
  success: (res) => {
    const result = JSON.parse(res.data);
    if (result.error) {
      // 处理错误
      console.error(result.message);
    } else {
      // 使用识别结果
      console.log('分类ID:', result.category_id);
      console.log('分类名称:', result.category_name);
    }
  }
});
```
